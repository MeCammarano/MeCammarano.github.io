Projects

Brand Storming

Brand Storming is a website developed with vanilla JavaScript and Node.js. The back end retrieves data using scraping techniques from various sources. The website provides a global view of a fashion brand: videos of advertising campaigns and parades, recent tweets, next events and stores.

MeteoWhere

MeteoWhere is a hybrid application developed using Ionic 3. It allows users to easily find places to do various activities based on the forecast. The app placed second at the App Challenge 2018, a competition held in Salerno Municipality.

Auction Mechanism

Auction Mechanism implements an auction based on a P2P network. It's possible to create new auctions, verify an auction's state, and offer money for an auction. It is developed using Apache Maven as a software project management and TomP2P.

Multispectral Image Analysis

The purpose is to analyze adjacent bands correlation of a multispectral image and the same bands' correlation of a multispectral image over time. We amended the SLSQ algorithm in order to introduce temporal redundancy.
